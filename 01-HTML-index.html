HTML পরিচিতি

HTML হলো ওয়েব পেজ তৈরির মূল ভিত্তি। প্রতিটি ওয়েবসাইট HTML দিয়ে গঠিত।

HTML কী?

HTML এর পূর্ণরূপ হলো HyperText Markup Language। এটি একটি মার্কআপ ল্যাঙ্গুয়েজ, প্রোগ্রামিং ল্যাঙ্গুয়েজ নয়। HTML ব্যবহার করে ওয়েব পেজের কাঠামো (structure) তৈরি করা হয়।

  • HyperText — লিংকের মাধ্যমে এক পেজ থেকে অন্য পেজে যাওয়া
  • Markup — ট্যাগ দিয়ে কন্টেন্টকে চিহ্নিত করা
  • Language — নির্দিষ্ট নিয়মে লেখার ভাষা

HTML কী কাজ করে?

একটি ওয়েবসাইটে তিনটি প্রধান প্রযুক্তি ব্যবহার হয়:

🏗️

HTML

কাঠামো তৈরি করে — হেডিং, প্যারাগ্রাফ, ছবি, লিংক

🎨

CSS

সাজসজ্জা করে — রঙ, ফন্ট, লেআউট

⚡

JavaScript

ইন্টারেক্টিভ করে — বাটন ক্লিক, অ্যানিমেশন

🌐

Browser

HTML Editors

HTML লেখার জন্য আপনার একটি Text Editor দরকার। অনেক ধরনের Editor আছে — বিনামূলে এবং পেইড।

জনপ্রিয় HTML Editors

HTML লেখার জন্য সবচেয়ে জনপ্রিয় Editors হলো:

  • VS Code — Microsoft-এর ফ্রি এডিটর, সবচেয়ে বেশি ব্যবহৃত
  • Notepad++ — হালকা ও দ্রুত, Windows-এর জন্য
  • Sublime Text — দ্রুত ও স্মার্ট এডিটর
  • Notepad — Windows-এর সাথে আসা, শুরু করার জন্য যথেষ্ট
💡সুপারিশ: শুরুতে VS Code ব্যবহার করুন। এটি ফ্রি, দ্রুত এবং অনেক সুবিধা আছে।

HTML Basic

HTML-এর বেসিক স্ট্রাকচার শিখলে যেকোনো ওয়েবসাইট তৈরি করা সহজ হয়ে যায়।

HTML ডকুমেন্ট স্ট্রাকচার

প্রতিটি HTML ডকুমেন্টের একটি নির্দিষ্ট কাঠামো থাকে:

index.html
<!DOCTYPE html>        <!-- HTML5 ডকুমেন্ট -->
<html lang="bn">          <!-- বাংলা ভাষা -->
  <head>
    <meta charset="UTF-8">     <!-- বাংলা সাপোর্ট -->
    <meta name="viewport" content="width=device-width">
    <title>পেজের নাম</title>
  </head>
  <body>
    <h1>মূল শিরোনাম</h1>
    <p>প্যারাগ্রাফ লেখা।</p>
  </body>
</html>

HTML ট্যাগ কী?

HTML ট্যাগ হলো ব্র্যাকেটের মধ্যে লেখা কীওয়ার্ড। বেশিরভাগ ট্যাগ জোড়ায় আসে:

ট্যাগের গঠন
<tagname> কন্টেন্ট এখানে </tagname>

<!-- উদাহরণ -->
<p>এটি একটি প্যারাগ্রাফ।</p>
<h1>বড় শিরোনাম</h1>

<!-- একা ট্যাগ (self-closing) -->
<br>      <!-- লাইন ব্রেক -->
<hr>      <!-- অনুভূমিক রেখা -->
<img>     <!-- ছবি -->

কমেন্ট লেখা

comment
<!-- এটি একটি কমেন্ট, ব্রাউজারে দেখা যায় না -->
<p>এটি প্যারাগ্রাফ।</p>
<!-- TODO: এখানে ছবি যোগ করতে হবে -->

HTML Elements

HTML Element হলো শুরু ট্যাগ + কন্টেন্ট + শেষ ট্যাগের সমন্বয়।

Element কী?

element structure
<p>আমি একটি প্যারাগ্রাফ।</p>
  ↑              ↑              ↑
শুরু ট্যাগ    কন্টেন্ট      শেষ ট্যাগ

Nested Elements (নেস্টেড)

একটি Element এর ভেতরে আরেকটি Element রাখাকে Nesting বলে:

nested
<body>
  <h1>শিরোনাম</h1>
  <p>
    এটি একটি প্যারাগ্রাফ, এখানে
    <strong>বোল্ড টেক্সট</strong> আছে।
  </p>
</body>

সাধারণ HTML Elements

Elementবিবরণউদাহরণ
<h1>–<h6>শিরোনাম<h1>শিরোনাম</h1>
<p>প্যারাগ্রাফ<p>লেখা</p>
<a>লিংক<a href="#">লিংক</a>
<img>ছবি<img src="photo.jpg">
<div>বিভাগ/কন্টেইনার<div>...</div>
<span>ইনলাইন কন্টেইনার<span>টেক্সট</span>
<br>লাইন ব্রেক<br>
<hr>অনুভূমিক রেখা<hr>

HTML Attributes

Attributes হলো HTML Element-এর অতিরিক্ত তথ্য, যা শুরু ট্যাগে লেখা হয়।

Attribute-এর গঠন

attribute syntax
<tagname attribute="value">কন্টেন্ট</tagname>

<!-- উদাহরণ -->
<a href="https://google.com">গুগলে যান</a>
<img src="photo.jpg" alt="ছবির বর্ণনা">
<p style="color: red;">লাল লেখা</p>

গুরুত্বপূর্ণ Attributes

Attributeকাজউদাহরণ
hrefলিংকের ঠিকানাhref="page.html"
srcছবি/ভিডিওর সোর্সsrc="image.jpg"
altছবির বিকল্প টেক্সটalt="ছবি নেই"
styleইনলাইন CSSstyle="color:red"
idইউনিক পরিচয়id="header"
classগ্রুপ পরিচয়class="btn"
targetলিংক কোথায় খুলবেtarget="_blank"
width/heightপ্রস্থ/উচ্চতাwidth="200"

HTML Paragraphs

<p> এই ট্যাগ দিয়ে Paragraph প্রকাশ করা হয়। <p> যদি কোন ডকুমেন্টকে এক বা একধিক প্যারাগ্রাফে প্রকাশ করতে চাই তখন এই ট্যাগ ব্যবহার করব।
paragraphs.html
<html>
<body>
  <p>এটি প্রথম প্যারাগ্রাফ। HTML স্বয়ংক্রিয়ভাবে ফাঁকা জায়গা সরিয়ে দেয়।</p>

<p>এটি দ্বিতীয় প্যারাগ্রাফ। প্যারাগ্রাফ আরো বড়ও হতে পারে।</p>

<!-- লাইন ব্রেক -->
<p>প্রথম লাইন।<br>দ্বিতীয় লাইন।</p>

<!-- অনুভূমিক রেখা -->
<hr>

pre tag
<pre>
  আমি     স্পেস দিয়ে
  লিখলাম।
</pre>

HTML Styles

internal CSS

  </style>

HTML Links

<a> ট্যাগ দিয়ে লিংক তৈরি হয়। href attribute-এ লিংকের ঠিকানা দিতে হয়।
links.html
<!-- বাইরের সাইটে লিংক -->
<a href="https://google.com">গুগল</a>

<!-- নতুন ট্যাবে খুলবে -->
<a href="https://google.com" target="_blank">নতুন ট্যাবে খুলুন</a>

<!-- একই পেজের ভেতরে -->
<a href="#section1">সেকশন ১-এ যান</a>

<!-- ইমেইল লিংক -->
<a href="mailto:info@example.com">ইমেইল করুন</a>

<!-- ফোন লিংক -->
<a href="tel:+8801234567890">ফোন করুন</a>

<!-- ছবিতে লিংক -->
<a href="page.html">
  <img src="logo.png" alt="লোগো">
</a>

Target Attribute

মানকাজ
_blankনতুন ট্যাব/উইন্ডোতে খুলবে
_selfএকই ট্যাবে খুলবে (ডিফল্ট)
_parentপ্যারেন্ট ফ্রেমে খুলবে
_topসম্পূর্ণ উইন্ডোতে খুলবে

HTML Images

<img> ট্যাগ দিয়ে ছবি যোগ করা হয়। এটি একটি self-closing ট্যাগ।
images.html
<!-- সাধারণ ছবি -->
<img src="photo.jpg" alt="একটি ছবি">

<!-- সাইজ সহ -->
<img src="photo.jpg" alt="ছবি" width="300" height="200">

<!-- ইন্টারনেট থেকে ছবি -->
<img src="https://example.com/img.png" alt="অনলাইন ছবি">

<!-- CSS দিয়ে স্টাইল -->
<img src="photo.jpg" style="border-radius: 50%; width: 100px;">

গুরুত্বপূর্ণ Attributes

Attributeকাজ
srcছবির পাথ বা URL (আবশ্যিক)
altছবি না দেখালে এই টেক্সট দেখায় (আবশ্যিক)
widthছবির প্রস্থ (pixel)

HTML Tables

টেবিল দিয়ে সারি ও কলামে তথ্য সাজানো যায়।
table.html
<table border="1">
  <thead>
    <tr>
      <th>নাম</th>
      <th>বয়স</th>
      <th>শহর</th>
    </tr>
  </thead>
  <tbody>
    <tr>
      <td>রাহেলা</td>
      <td>২৫</td>
      <td>ঢাকা</td>
    </tr>
    <tr>
      <td>করিম</td>
      <td>৩০</td>
      <td>চট্টগ্রাম</td>
    </tr>
  </tbody>
</table>

টেবিল Tags

ট্যাগকাজ
<table>টেবিল শুরু করে
<thead>টেবিলের হেডার অংশ
<tbody>টেবিলের মূল তথ্য অংশ

<th>Table Header Cell — বোল্ড ও মাঝে
<td>Table Data Cell — সাধারণ ঘর
colspanকয়েকটি কলাম জুড়ে
rowspanকয়েকটি সারি জুড়ে

HTML Lists

HTML-এ তিন ধরনের লিস্ট আছে: Unordered, Ordered এবং Description List।
lists.html
<!-- বুলেট লিস্ট -->
<ul>
  <li>আম</li>
  <li>কাঁঠাল</li>
  <li>লিচু</li>
</ul>

<!-- নম্বর লিস্ট -->
<ol>
  <li>প্রথম ধাপ</li>
  <li>দ্বিতীয় ধাপ</li>
  <li>তৃতীয় ধাপ</li>
</ol>

<!-- বর্ণনা লিস্ট -->
<dl>
  <dt>HTML</dt>
  <dd>ওয়েব পেজের কাঠামো তৈরি করে</dd>
  <dt>CSS</dt>
  <dd>ওয়েব পেজ সাজায়</dd>
</dl>
আউটপুট
Unordered:
  • আম
  • কাঁঠাল
  • লিচু
Ordered:
  1. প্রথম ধাপ
  2. দ্বিতীয় ধাপ
  3. তৃতীয় ধাপ

HTML Video

HTML5 থেকে <video> ট্যাগ দিয়ে সহজেই ভিডিও যোগ করা যায়।
video.html
<!-- সাধারণ ভিডিও -->
<video width="400" controls>
  <source src="video.mp4" type="video/mp4">
  আপনার ব্রাউজার ভিডিও সাপোর্ট করে না।
</video>

<!-- অটোপ্লে ও মিউটেড -->
<video autoplay muted loop width="400">
  <source src="video.mp4" type="video/mp4">
</video>

<!-- YouTube Embed -->
<iframe width="560" height="315"
  src="https://www.youtube.com/embed/VIDEO_ID"
  allowfullscreen>
</iframe>

Video Attributes

Attributeকাজ
controlsপ্লে, পজ, ভলিউম বাটন দেখায়
autoplayপেজ লোড হলে স্বয়ংক্রিয়ভাবে চালু হয়
mutedশব্দ বন্ধ থাকে
loopবারবার চলতে থাকে
posterভিডিও চালুর আগে থাম্বনেইল

HTML Forms

Form দিয়ে ব্যবহারকারী থেকে তথ্য নেওয়া হয়। লগইন, রেজিস্ট্রেশন সব ফর্ম দিয়ে তৈরি।
form.html
<form action="submit.php" method="POST">

  <!-- টেক্সট ইনপুট -->
  <label for="name">নাম:</label>
  <input type="text" id="name" name="name" placeholder="আপনার নাম">

  <!-- ইমেইল -->
  <label for="email">ইমেইল:</label>
  <input type="email" id="email" name="email">

  <!-- পাসওয়ার্ড -->
  <label>পাসওয়ার্ড:</label>
  <input type="password" name="pass">

  <!-- টেক্সট এরিয়া -->
  <label>বার্তা:</label>
  <textarea name="msg" rows="4"></textarea>

  <!-- সাবমিট বাটন -->
  <button type="submit">পাঠান</button>

</form>

HTML Input Types

<input> ট্যাগে type attribute দিয়ে বিভিন্ন ধরনের ইনপুট তৈরি করা যায়।
input-types.html
<input type="text">          <!-- সাধারণ টেক্সট -->
<input type="email">         <!-- ইমেইল ভেলিডেশন -->
<input type="password">      <!-- পাসওয়ার্ড -->
<input type="number">        <!-- নম্বর -->
<input type="tel">           <!-- ফোন নম্বর -->
<input type="date">          <!-- তারিখ নির্বাচন -->
<input type="checkbox">      <!-- চেকবক্স -->
<input type="radio">         <!-- রেডিও বাটন -->
<input type="file">          <!-- ফাইল আপলোড -->
<input type="range">         <!-- স্লাইডার -->
<input type="color">         <!-- কালার পিকার -->
<input type="submit">        <!-- সাবমিট বাটন -->
<input type="reset">         <!-- রিসেট বাটন -->
<input type="hidden">        <!-- লুকানো মান -->

Form Elements

ফর্মে input ছাড়াও আরো অনেক উপাদান ব্যবহার করা যায়।
form-elements.html
<!-- Dropdown Select -->
<select name="city">
  <option value="dhaka">ঢাকা</option>
  <option value="ctg">চট্টগ্রাম</option>
  <option value="sylhet">সিলেট</option>
</select>

<!-- Datalist (সার্চ সাজেশন) -->
<input list="fruits" placeholder="ফল খুঁজুন">
<datalist id="fruits">
  <option value="আম">
  <option value="কলা">
  <option value="লিচু">
</datalist>

<!-- Fieldset (গ্রুপ) -->
<fieldset>
  <legend>ব্যক্তিগত তথ্য</legend>
  <input type="text" placeholder="নাম">
  <input type="email" placeholder="ইমেইল">
</fieldset>
