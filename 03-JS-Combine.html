Lecture 12

Conditional / Condition কী?

Definition

Condition মানে হলো শর্ত

JavaScript-এ condition ব্যবহার করে সিদ্ধান্ত নেওয়া হয়

📌 বাস্তব উদাহরণ:
  • যদি বৃষ্টি হয় → ছাতা নিব
  • না হলে → ছাতা নিব না

এই “যদি–নাহলে” লজিকটাই Conditional

01-Comparison Operators (Condition বানাতে লাগে)

  Operators          Meaning
      >  --------  Greater then
      <  --------  Less then
      <=  --------  Greater than or equal to
      =<  --------  Less than  or equal to
      ==  --------  Equal to
      ===  -------- Equal value and equal type
      !=  --------  Not equal
      !==  -------- Not equal Value or not equal type
Example:
10 == "10" //true
10 === "10" //false

⚠️ মনে রাখো: সব সময় Equal value and equal type === ব্যবহার করো, Equal to == নয়।

2. JS if Statement - যদি

If statement দিয়ে condition check করে code execute করা হয়।

  • Condition Check
  • True হলে Execute
  • False হলে Skip
Syntax
if (condition) {
  //true হলে
   }

If-statements.js
let age = 20;     // যদি বয়স 18 বা বেশি হয়
if (age >= 18) {
  console.log ("তুমি ভোট দিতে পারবে");
  }

একাধিক Conditions
let marks = 85;
if (marks >= 80) {
  console.log ("A+ গ্রেড পেয়েছ!");
  console.log ("অভিনন্দন!");
  }

03-JS if else - যদি - না হলে

Syntax
if (condition) {
      // true হলে
  } else {
      // false হলে
  }

Condition true না হলে else block execute হয়।

If-else.js
let temperature = 35;
let (temperature > 30) {
     console.log ("গরম পড়েছে! ☀️");
} else {
      console.log ("আবহাওয়া ভালো আছে 🌤️")
}

একটা ব্লক চলবে, দুটো ব্লক একসাথে কখনই চলবে না।

02-Else-If - একাধিক শর্ত

একাধিক conditions check করার জন্য else-if ব্যবহার হয়।

syntax

if (condition1){
} else if (condition2) {
} else {
}

যখন একাধিক শর্ত থাকে, তখন যদি শর্ত সত্য না হয়, তাহলে else এর কোড চলবে।

else-if.js

let marks = 72;
let (marks >= 80) {
    console.log ("গ্রেড: A+ 🌟");

} else if (marks >= 70) {
    console.log ("গ্রেড: A+ 🌟");

} else if (marks >= 60) {
    console.log ("গ্রেড: B ✨");

} else if (marks >= 50) {
    console.log ("গ্রেড: C 💫");

} else { console.log ("গ্রেড: F (ফেল) 😔");
}
  • উপর থেকে নিচে চেক করে
  • প্রথম true পেলেই থেমে যায়

JS Ternary Operator (Shortcut if else)

Ternary operator হলো if-else এর সংক্ষিপ্ত রূপ। একটা লাইনে লেখা যায়।

Normal If-else Syntax
if (condition) {
  // true হলে
} else {
  // false হলে
}

if (age >= 18) {
  /status = "প্রাপ্তবয়স্ক";
} else {
  status = "অপ্রাপ্তবয়স্ক";
}

Ternary Syntax

condition ? trueValue : falseValue

// Ternary দিয়ে একই কাজ (একটা লাইনে!)
let age = 20;
if status  >= 18 ? "প্রাপ্তবয়স্ক" : "অপ্রাপ্তবয়স্ক";
    console.log (status);

Price discount example
let price  = 500;
let discount  = price > 1000 ? 100 : 50;
console.log ( "Discount: " + discount ) //"Discount: 50"

বাস্তব উদাহরণ:

JavaScript লগইন চেক
let username   = "আহমেদ";
let password   = "১২৩৪";

let (username  === "আহমেদ" && password === "১২৩৪") {
    console.log ( "লগইন সফল!");
} else {
    console.log ( "ভুল তথ্য");
}

Lecture 15

Switch Statement - একাধিক Case

যখন একই variable এর অনেক values check করতে হয়, তখন switch ব্যবহার করা ভালো।

switch.js
let day = 4;
let dayName;
switch  (day) {    
    case 1: dayName = "শনিবার"; break;
    case 2: dayName = "রবিবার"; break;
    case 3: dayName = "সোমবার"; break;
    case 4: dayName = "মঙ্গলবার"; break;
    case 5: dayName = "বুধবার"; break;
    case 6: dayName = "বৃহস্পতিবার"; break;
    case 7: dayName = "শুক্রবার"; break;
    default : dayName = "ভুল দিন!";
}
        console.log (dayName); // "বুধবার"
    ✅ Switch এর গুরুত্বপূর্ণ অংশ:
  • case - প্রতিটা option
  • break - Switch থেকে বের হওয়া (না থাকলে পরের case ও চলবে!)
  • efault - কোনো case match না করলে (else এর মতো)
switch-multiple.js
let fruit = "আম";
let color;
switch  (fruit) {
    case "কলা" : case "আম" : color = "হলুদ"; break;
    case "আপেল" : case "স্ট্রবেরি" : color = "লাল"; break;
    case "কমলা" : color = "কমলা"; break;
    default : color = "জানি না";
}
        console.log ("রঙ:") + color); // "রঙ: হলুদ"

বিষয়                  If-Else                        Switch
ব্যবহার            যেকোনো condition           একই variable এর বিভিন্ন values
Comparison       Range check করা যায় (>, <)   শুধু equality (===)
Readability      অনেক conditions হলে জটিল   অনেক cases হলে সুন্দর
Performance      কম cases এ ভালো             বেশি cases এ faster
Multiple Conditions  &&, || ব্যবহার করা যায়      Multiple cases একসাথে
    🔑 Beginner Tips (খুব গুরুত্বপূর্ণ)
  • ✔️ Always use ===
  • ✔️ Code readable রাখো
  • ✔️ Nested if কম ব্যবহার করো
  • ✔️ Logic আগে ভাবো, পরে কোড লেখো

Lecture 17

Logical Operators যুক্তি

একাধিক conditions একসাথে check করার জন্য logical operators ব্যবহার হয়।

তিনটা Logical Operators:

  • AND (&&)
  • OR (||)
  • NOT (!)
AND (&&)
সব condition true হতে হবে
if (age >= 18 && age <= 30) 

OR (||)
যেকোনো একটা true হলেই চলবে
if (age >= 18 || age > 60) 

NOT (!)
উল্টো করে দেয়- True হলে false রেজাল্ট দিবে আর false হলে true রেজাল্ট দিবে।
if (!isLoggedIn) 

// AND (&&) - দুইটাই সত্য হতে হবে
if (age >= 18 && hasTicket) {
    console.log ("সিনেমা দেখতে পারবে");
}

// OR (||) - যেকোনো একটা সত্য হলেই হবে
if (age < 12 || age > 60) {
    console.log ("ছাড় পাবে");
}

// NOT (!) - উল্টা করে
if (!hasTicket) {
    console.log ("টিকেট কিনতে হবে");
}
