CSS পরিচিতি

CSS হলো Cascading Style Sheets — এটি HTML পেজকে সুন্দর ও আকর্ষণীয় করে তোলার ভাষা।

CSS কী?

CSS (Cascading Style Sheets) একটি স্টাইলশিট ভাষা যা HTML এলিমেন্টের চেহারা নির্ধারণ করে। CSS দিয়ে আপনি রং, ফন্ট, সাইজ, স্পেসিং, লেআউট এবং আরো অনেক কিছু নিয়ন্ত্রণ করতে পারেন।

সহজ ভাষায় বলতে গেলে — HTML হলো ওয়েবপেজের কাঠামো, আর CSS হলো সেই কাঠামোর সাজসজ্জা।

CSS কেন শিখবেন?

  • ওয়েবপেজকে দৃষ্টিনন্দন করা যায়
  • সব ডিভাইসে সুন্দরভাবে দেখানো যায় (Responsive Design)
  • HTML থেকে ডিজাইন আলাদা রাখা যায়
  • একটি CSS ফাইল দিয়ে পুরো ওয়েবসাইট স্টাইল করা যায়

CSS এর একটি সহজ উদাহরণ

CSS উদাহরণ
h1 {
  color: blue;
  font-size: 24px;
  text-align: center;
}
💡 মনে রাখুন: CSS শেখা HTML এর পরের ধাপ। আগে HTML ভালো করে শিখুন, তারপর CSS।

CSS Syntax

CSS-এর নিজস্ব একটি নিয়ম আছে — কীভাবে লিখতে হয় তা জানা জরুরি।

CSS Syntax কী?

CSS Syntax হলো CSS লেখার সঠিক নিয়ম। একটি CSS রুল তিনটি অংশে তৈরি — Selector, Property এবং Value।

CSS Syntax কাঠামো
selector {
  property: value;
}

উদাহরণ

উদাহরণ
p {
  color: red;
  font-size: 16px;
  line-height: 1.6;
}

CSS কোথায় লেখা যায়?

  • Inline CSS: সরাসরি HTML ট্যাগের ভেতরে style অ্যাট্রিবিউট দিয়ে
  • Internal CSS: HTML ফাইলের <style> ট্যাগের ভেতরে
  • External CSS: আলাদা .css ফাইলে লিখে লিঙ্ক করে
তিন ধরনের CSS
/* Inline CSS */
<p style="color: blue">এটি Inline CSS</p>

/* Internal CSS */
<style>
  p { color: green; }
</style>

/* External CSS */
<link rel="stylesheet" href="style.css">
💡 সুপারিশ: সবসময় External CSS ব্যবহার করুন। এটি সবচেয়ে সুবিধাজনক এবং পরিষ্কার পদ্ধতি।

CSS Selectors

Selector দিয়ে বলা হয় কোন HTML এলিমেন্টে স্টাইল প্রয়োগ হবে।

Selector কী?

CSS Selector হলো এমন একটি প্যাটার্ন যা নির্দিষ্ট HTML এলিমেন্ট(গুলো) বেছে নেয়, যেন সেগুলোতে স্টাইল দেওয়া যায়।

সাধারণ Selectors

  • Element Selector: যেমন p, h1, div — সরাসরি ট্যাগ নাম
  • Class Selector: .classname — ডট দিয়ে শুরু
  • ID Selector: #idname — হ্যাশ দিয়ে শুরু
  • Universal Selector: * — সবকিছু সিলেক্ট করে
Selector উদাহরণ
/* Element Selector */
p { color: black; }

/* Class Selector */
.highlight { background: yellow; }

/* ID Selector */
#header { font-size: 24px; }

/* Universal Selector */
* { margin: 0; padding: 0; }
💡 টিপস: Class Selector সবচেয়ে বেশি ব্যবহৃত হয়। ID সাধারণত JavaScript-এর জন্য রাখা হয়।

CSS Colors

CSS-এ রং নির্ধারণ করার বিভিন্ন পদ্ধতি আছে — নাম, HEX, RGB, HSL।

রং লেখার পদ্ধতি

  • Color Name: red, blue, green ইত্যাদি নাম দিয়ে
  • HEX: #ff0000 — হেক্সাডেসিমাল কোড
  • RGB: rgb(255, 0, 0) — লাল, সবুজ, নীলের মান
  • HSL: hsl(0, 100%, 50%) — Hue, Saturation, Lightness
  • RGBA: rgba(255, 0, 0, 0.5) — Alpha/Transparency সহ
রং ব্যবহারের উদাহরণ
h1 { color: red; }
h2 { color: #3498db; }
p  { color: rgb(46, 204, 113); }
.box { background-color: rgba(0,0,0,0.3); }
💡 টিপস: ডিজাইনে HEX কোড সবচেয়ে বেশি ব্যবহৃত হয়। Transparency দরকার হলে RGBA ব্যবহার করুন।

CSS Background

Definition

CSS background ব্যবহার করা হয় কোনো element-এর পেছনের color, image, position, size, repeat, scroll behavior ইত্যাদি control করার জন্য।

All Background Properties

  • background-color
  • background-image
  • background-repeat
  • background-attachment
  • background-position
  • background-size
  • background- (shorthand property)

1. background-color

ব্যাকগ্রাউন্ডে রঙ দেয়

div {
    background-color: red;
    }
📌 Values:
  • color name → red, blue
  • HEX → #ff0000
  • RGB → rgb(255,0,0)

🖼️ 2. background-image

👉 ব্যাকগ্রাউন্ডে ছবি বসায়

div {
    background-image: url("image.jpg");
    }

🔁3. background-repeat

👉 ছবি repeat হবে কিনা সেট করে

div {
    background-repeat: no-repeat;
    }
📌 Values:
  • repeat (default)
  • no-repeat
  • repeat-x
  • repeat-y

📍4. background-position

👉 ছবি কোথায় থাকবে সেট করে

div {
    background-position: center;
   }
📌 Values:
  • top
  • bottom
  • left
  • right
  • center
  • Custom → 50% 50%

📏 5. background-size

👉 ছবির সাইজ নিয়ন্ত্রণ করে

div {
    background-size: cover;
   }
📌 Values:
  • cover → পুরো area cover করে
  • contain → পুরো ছবি দেখায়
  • Custom → 100px 200px

📌 6. background-attachment

👉 scroll behaviour নিয়ন্ত্রণ করে

div {
    background-posattachmentition: fixed;
   }
📌 Values:
  • scroll (default)
  • fixed (parallax effect)
  • local

📦 7. background-clip

👉 background কোথায় পর্যন্ত যাবে

div {
    background-clip: border-box;
   }

🧩 8. background-origin

👉 background কোথা থেকে শুরু হবে

div {
    background-origin: padding-box;
   }

⚡ 9. Shortcut Property (Most Important)

👉 সব একসাথে লেখার জন্য background

div{
    background: red url("image.jpg") no-repeat center/cover;
   }
📌 Order:
  • color → image → repeat → position / size → attachment

🎨 10. Gradient Background

👉 CSS দিয়ে gradient বানানো যায়

div{
    background: linear-gradient(to right, red, blue);
   }
📌 Types::
  • linear-gradient()
  • radial-gradient()

CSS Table

Definition

HTML <table> element-এর চেহারা এবং layout CSS দিয়ে সম্পূর্ণ নিয়ন্ত্রণ করা যায়।

1. Table-এর মূল HTML গঠন

<table>
  <thead>
    <tr>
      <th> নাম </th>
      <th> বয়স </th>
      <th> শহর </th>
    </tr>
  </thead>

  <tbody>
    <tr>
      <td> রাহেলা </td>
      <td> 24 </td>
      <td> ঢাকা </td>
    </tr>
  </tbody>

  <tfoot>
    <tr>
      <td colspan= "3"> মোট: ১ জন </td>
    <tr>
  </tfoot>
</table>

2. border-collapse — সবচেয়ে গুরুত্বপূর্ণ property

table {
border-collapse: url("image.jpg"); /* default — দুটো আলাদা border */ 
border-collapse: url("image.jpg"); /* একটাই border, সবচেয়ে বেশি ব্যবহৃত */ 
      }
📌 Note:

Collapse ছাড়া <td> এবং <th> -এর border আলাদা আলাদা দেখায় — দ্বিগুণ মোটা লাগে।

3. Border-spacing

border-collapse: separate থাকলে cell-গুলোর মধ্যে ফাঁক দেওয়া যায়:

table {
        border-collapse: separate;
        border-spacing: 10px; /* চারদিক সমান */
        border-spacing: 10px 5px;/* horizontal vertical */
        }

📍4. Width এবং Height

div { width: 100%;       /* পুরো container জুড়ে */
th, td { height: 48px;    /* row-এর উচ্চতা */
th, td { width: 200px;   /* column-এর প্রস্থ */

5. Table Layout

table { table-layout: auto;   /* default — content অনুযায়ী width ঠিক হয় */
table { table-layout: fixed;   /* width নির্দিষ্ট, content overflow হয় */
📌 Note:

fixed দিলে table দ্রুত render হয় এবং column width নিয়ন্ত্রণ সহজ হয়।

6. Text-align এবং vertical-align

    th { text-align: left;        /* header বাম দিকে */
    td { text-align: center;     /* data মাঝে */

    td { vertical-align: top;     /* উপরে */
    td {  vertical-align: middle;  /* মাঝে — default */
    td { vertical-align: bottom;  /* নিচে */

7. Zebra Striping — alternate row color

👉 background কোথায় পর্যন্ত যাবে

tbody tr:nth-child(even) {
    background-color; #f5f5f5;
    }

tbody tr:nth-child(odd) {
    background-color; #ffffff;
    }

8. Hover effect

tbody tr:hover {
    background-color: #e8f4fd;;
    cursor: pointer;
    }

9. Responsive Table

ছোট screen-এ table overflow হয়।

সমাধান:

.table-wrapper {
    overflow-x: auto;     /* horizontal scroll আসবে */
    -webkit-overflow-scrolling: touch;
    }

<div class"table-wrapper">
  <table>.......</table> 
</div>

10. Caption

Table-এর শিরোনাম দেওয়া যায়:

caption{
   caption-side: top; /* উপরে — default */
   caption-side: bottom;  /* নিচে */
   font-weight: bold;
   margin-bottom: 8px;
  }

11. একটি সম্পূর্ণ সুন্দর Table-এর উদাহরণ

    overflow-x: auto;
    border-radius: 12px;
    border: 1px solid #e5e7eb;
    }

    width: 100%;

    }
